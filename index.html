Easma

Easma is cms project which is coding with php. It is simple cms example. No framework,No template engine !

You can make your blog or website with Easma and you can join me for developing it.

How to install

  • Copy all files to under httpdocs/ or public_html/
  • Import .sql to mysql
  • Edit config.php
  • Finally you are ready to go. Go to http://$base/panel

Example

Let's make General Options page for web site

/panel/views/menu.php we add this code

  • $site->menu("Site Options","general_options");

Site Options is our menu name. general_options is our php view script which is under /panel/views/ folder.

Creating general_options.php

Create general_options.php to /panel/views/general_options.php and Add this lines...

  • if (!isset($base)){ exit(); }

  • $site->check_admin();

  • $site->post_update("ok","general","1","Site Options");

// update general table where id=1

  • $site->select_single_query("general","where id='1'");

//check table's data

  • $site->view->form("POST","",0);

// 0 means no file post

  • $site->view->varchar("inpt2","title","Web site Title :",stripslashes($site->view_query['title']));

// inpt2 is css class name. title is table column.

varchar, add_pic, ckeditor etc. for check /panel/controller/form_view.php

  • $site->view->varchar("inpt2","description","Web site description :",stripslashes($site->view_query['description']));

  • $site->view->varchar("inpt2","keywords","Web site keywords :",stripslashes($site->view_query['keywords']));

  • $site->view->varchar("inpt2","facebook","Facebook :",stripslashes($site->view_query['facebook']));

  • $site->view->varchar("inpt2","twitter","Twitter :",stripslashes($site->view_query['twitter']));

  • $site->view->add_pic("pic_add","logo","Logo : ","general","1","Picture");

  • $site->view->ckeditor("text","content","Main Text :",stripslashes($site->view_query['content']),$site->site_url);

  • $site->view->ckeditor("text","footer_text","Footer :",stripslashes($site->view_query['footer_text']),$site->site_url);

  • $site->view->submit("submit","ok","Update");

  • $site->view->form_close();

And general_options ready
